# Pokédex App - Quick Start

## 🎯 5-Minute Setup

### Step 1: Clone or Download
```bash
git clone https://github.com/saheedd33/pokedex.git
cd pokedex
```

### Step 2: Open the App
**Option A - Direct (Easiest):**
- Right-click `index.html`
- Open with your browser

**Option B - Local Server:**
```bash
# Python
python3 -m http.server 8000

# Or Node.js
npx http-server

# Then visit: http://localhost:8000
```

### Step 3: Sign Up & Play
1. Create new account
2. Search for Pokémon
3. Add to favorites
4. Build your team
5. Check type coverage
